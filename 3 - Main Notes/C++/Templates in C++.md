# Templates in C++

**Definition**  
Templates allow writing generic code that works with any data type, enabling **compile-time polymorphism**.

**Purpose**  
Write once, use with multiple types – avoid code duplication while keeping type safety.

## Core Concept

Templates are like **blueprints** that generate specific code for each data type at compile time.
